# 70 — Terminology & Naming
## Principle
Use one term for one concept. Never switch synonyms mid-flow.
## Rules
- If the system calls it a “ride,” don’t call it a “trip” elsewhere.
- Keep labels short and plain.
- Avoid internal jargon.
## Recommended core terms (starter)
- Ride
- Driver
- Rider
- Offer
- Counter
- Confirm
- Active
- Completed
