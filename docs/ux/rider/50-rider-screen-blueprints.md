# 50 — Rider Screen Blueprints
## Template A — Map + Sheet (Home)
- Map: passive context.
- Sheet (collapsed): “Where to?”
- Sheet (peek): route summary + anchor price.
- Sheet (expanded): route edit.
## Template B — Estimate / QR Session
- Sheet (peek): anchor price + route.
- Primary CTA: Generate QR.
- Expanded view shows QR full-size.
## Template C — Discovery / Offers
- Same container; different states:
	- Searching
	- Offers received
	- No drivers
	- Expanding search
- Offers list: cards.
- Primary action: select offer (card).
## Template D — Hold / Active
- Hold: driver summary, status, confirm.
- Active: driver info + agreed fare + safety/report entry.
