# 51 — Rider Critical States Layout
## Offers state
- Header: state label (Offers received)
- Price anchor: hidden unless user selects a card (or show per card)
- List: top 3 cards + “+X more”
- Secondary: expand search (if needed)
## Hold state
- Header: Waiting / Driver selected
- Driver summary card
- Primary: Confirm
- Secondary: Change selection
## Active state
- Header: Active
- Agreed fare (single number)
- Driver identity + verification
- Route summary
- Primary: Show QR / details
- No cancel
