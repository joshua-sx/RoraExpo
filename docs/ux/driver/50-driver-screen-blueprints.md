# 50 — Driver Screen Blueprints
## Template A — Driver Home (On duty)
- Top: On duty toggle.
- Middle: state card (waiting / incoming request / selected).
- Bottom: minimal nav (history, profile, support).
## Template B — Incoming Request (Card)
Card includes:
- Route summary
- Anchor price
- Pickup distance/time
Actions:
- Primary: Accept
- Secondary: Counter
- Tertiary: Decline
## Template C — Counter Flow (Fast)
- Show anchor price.
- Preset increments.
- Optional custom amount.
- Primary: Send counter.
## Template D — Selected / Active
- Selected: navigate to pickup + QR scan instructions.
- Active: status + complete ride CTA.
