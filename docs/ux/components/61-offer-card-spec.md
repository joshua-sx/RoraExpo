# 61 — Offer Card Spec
## Required fields
- Price (primary)
- Driver name/photo (if available)
- Verification badges
- Pickup distance/time
## Optional fields
- “Good deal / Pricier than usual” tag
- Vehicle details
## Layout rules
- Price is the most visually prominent element.
- Trust badges are visible but smaller than price.
- Tap selects the offer.
## Forbidden
- Dense breakdown tables.
- Multiple primary buttons inside the card.
